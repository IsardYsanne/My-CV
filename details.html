Меню «Бургер»
2021
Написан при помощи JavaScript, с использованием команд DOM: при помощи поиска по селектору классов ".icon-menu" и .menu__body, а также toggle/remove. Кроме того, использовался обработчик событий, с помощью которого сделана активация меню бургера, при открытии - появляется крестик, при нажатии на него меню закрывается. Добавлена анимация.
Форма
2021
Простые формы, сделанные при помощи html-тегов: form, input, textarea. Отправка и обработка формы осуществляется при помощи php, взаимодействие с которым осуществляется через серверную платформу Open Server panel, для переноса собственных проектов и внедрения PHP в верстку.
